# Github-Blog mit Jekyll

## Local dev
+ Dev-Server starten mit `bundle exec jekyll serve`
+ erreichbar unter `http://localhost:4000`

## Styling
Eigene Styles werden unter `_sass/minima/custom-styles.scss` eingetragen.
